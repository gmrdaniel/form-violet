Detalles del layout de desktop

  • Canvas limitado: 440px para wizard, 560px para scroll (S2/S3). Todo centrado horizontalmente.
  • Fondo con grid punteado muy tenue + gradient radial sutil violeta arriba. No distrae, agrega profundidad.
  • Selector de idioma ES · EN arriba a la derecha (aparece sólo cuando i18n esté conectado).
  • Check "✓ Guardado" arriba a la izquierda — aparece 1s tras escribir y se desvanece.
  • Banner de continuar aparece solo si hay draft válido (< 7 días). 2 botones: empezar de cero o continuar.
  • Móvil: mismos containers pero full-width con padding 16px. El fondo con grid y gradient se esconde en <768px.
